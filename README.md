Java Drawing App 🎨

A simple and interactive painting application built with Java Swing.

This project demonstrates the power of Java for creating graphical user interfaces (GUI) and handling user input events. Users can draw freely on a canvas, choose colors, and clear the screen.

Features

Draw with mouse: Click and drag to create freehand drawings.

Color selection: Switch between black, red, and blue.

Clear canvas: Reset the drawing area at any time.

Simple, intuitive GUI: Uses Java Swing for a responsive interface.
